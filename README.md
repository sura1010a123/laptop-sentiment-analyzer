# 💻 Laptop Review Sentiment Analyzer

This project is a **Sentiment Analysis Web App** that predicts whether a laptop review is **Positive, Neutral, or Negative** using a deep learning model.

The application uses a **BiLSTM neural network** trained on laptop review data and provides predictions through a **Streamlit web interface**.

---

## 🚀 Features

- Predicts sentiment of laptop reviews
- Supports **Positive, Neutral, and Negative** classification
- Simple **web interface using Streamlit**
- Uses **Deep Learning (BiLSTM)** for prediction

---

## 🧠 Technologies Used

- Python
- TensorFlow / Keras
- NumPy
- Streamlit
- Natural Language Processing (NLP)
